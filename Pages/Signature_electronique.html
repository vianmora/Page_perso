Créer sa propre signature électronique

Voici la signature que j'utilise lorsque j’envoie des mails pour le FOCEEN. Le mieux c’est encore d’avoir sa propre signature personnalisée avec le logo de son association ;)
Je vais donc ici tenter de vous montrer comment faire :D

3) Vous voulez changer l’image à gauche ?

Pas de soucis :D
Comme vous pouvez le comprendre, les personnes auquels vous enverrez vos mail n’auront pas accès à votre ordinateur, il est donc impératif d’héberger toutes ces images sur votre site perso. Et ensuite indiquer dans src l’adresse absolue de votre image

Coment faire :/

Vous avez tous un site perso fourni par l’école ;) et si vous ne voulez pas utiliser le vôtre, votre association en a un également, donc commençons.

Normalement si vous avez suivi le cours dédié au S5 vous savez que votre page se trouve à l’adresse suivante : [initiale prénom][nom].perso.centrale-marseille.fr. Pour le modifier il vous faut télécharger un client SFTP (je vous recommande Filezilla) qui vous donnera accès au dossier qui vous est réservé sur les serveurs de l’école.

Une fois que vous l’avez téléchargé, vous accédez à votre domaine en utilisant les paramètres suivant :
--> Nom d’hôte : sas1.centrale-marseille.fr
--> Nom d’utilisateur et mot de passe : ceux que vous utilisez normalement à centrale

Voilà, créez maintenant un dossier « signature », mettez y votre image : monimage.png et votre image est désormais disponible à l’adresse : "[initiale prénom][nom].perso.centrale-marseille.fr/signature/monimage.png"

Ainsi ligne 8, dans src, vous pouvez modifier l’adresse en remplaçant la mienne par la vôtre.

!!!!pensez à modifier la taille également pour ne pas vous retrouver avec quelque chose de disproportionné ;)

Astuce si vous voulez créer votre propre signature

Vous l’avez vu, une signature électronique est en réalité du code HTML couplé à du CSS.Cependant, les clients de messagerie ne sont pas des navigateurs et peuvent ne pas comprendre toutes vos instructions. Ainsi tout d’abord, une signature doit être un unique fichier .html où vous insèrerez tout votre code.

Pour cela, quand vous aurez créé votre tableau, et remplit avec du texte et des images, il vous faudra alors le mettre en forme.

Pour cela dans les balises adaptées, utilisez l’attribut « style ». Dans cet attribut, vous pouvez y mettre toutes les instructions en CSS et elles seront reconnues

Attention : n'utilisez pas d’attributs spécifiques du style border, height, width directement dans la balise, préférez les utilisez dans l’attribut style en langage CSS pour garantir l’optimalité.

Quelques exemples :
  • style = "width : 32px ; height : 32px;"
    --> Définit la taille de mon objet de 32px X 32px
  • style="color:#1a3745; padding : 10px;"
    --> Indique que mon objet sera de couleur bleue avec une indentation de 10px

Aussi, n’oubliez pas les ";" à la fin du code CSS sinon il risque de ne pas être reconnu non plus.

Un autre souci, c’est que certain clients reconnaissent et affichent correctement la signature, mais lorsqu’ils vous répondent, cette signature n’a plus la taille désirée. Pas de panique, ceci est dut à leur client de messagerie. Eux n’ont pas vu le changement seul vous recevez un mail avec cette signature.Si vous lui répondez, supprimez simplement l’ancienne ;)

intégration dans le client de messagerie

On arrive à la fin :D la dernière étape est d’insérer notre code dans notre gestionnaire de mail ;)

attention : le client de messagerie de l’ENT ne gère pas les signatures, il est donc impératif de télécharger un client annexe soit Thunderbird soit Mail pour mac ;)

    Pour Thunderbird donc :
  • Ouvrez un nouveau document texte (bloc note)
  • Copiez y le code html d’htmlonline
  • Enregistrez le sur votre ordinateur dans un endroit où vous ne le supprimerez pas ;)
  • Rendez vous dans thunderbird
  • Menu -> Option -> Paramètre de Compte
  • Allez dans la boite mail voulue mais vous pouvez apposer la même signature sur chacun de vos comptes
  • Cochez la case « aposer la signature à partir d’un fichier »
  • Recherchez votre fichier html et voilà ;)
